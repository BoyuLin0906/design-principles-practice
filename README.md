# Design Principles Practice

This repository is a place to practice design principles in a simple, hands-on way. The goal is to keep each example small, clear, and easy to revisit.

The project currently uses Python with `uv` for environment and dependency management.

Each principle should have its own directory, usually with:

- `README.md`
- `example.py`
- `bad_example.py`

This keeps every principle easy to compare, discuss, and extend over time.
